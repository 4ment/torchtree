class torchtree.distributions.transforms.TrilExpDiagonalTransform(cache_size=0)[source]#

Bases: torch.distributions.Transform

Transform a 1D tensor to a triangular tensor.

The diagonal of the triangular matrix is exponentiated. Useful for variational inference with the multivariate normal distribution as the variational distribution and it is parameterized with scale_tril, a lower-triangular matrix with positive diagonal.

Example

>>> x = torch.tensor([1., 2., 3.])
>>> y = TrilExpDiagonalTransform()(x)
>>> y
tensor([[ 2.7183,  0.0000],
        [ 2.0000, 20.0855]])
>>> torch.allclose(TrilExpDiagonalTransform().inv(y), x)
True

class torchtree.distributions.transforms.ConvexCombinationTransform(weights: torchtree.core.abstractparameter.AbstractParameter, cache_size=0)[source]#

Bases: torch.distributions.Transform

Transform via the mapping \(y = \frac{x}{\sum_{i=1}^K \alpha_i x_i}\).

The transformation satisfies \(\sum_{i=1}^K \alpha_i y_i = 1\) where \(\alpha_i \geq 0\) and \(\sum_{i=1}^K \alpha_i = 1\).

class torchtree.distributions.transforms.LinearTransform(weight: torchtree.core.abstractparameter.AbstractParameter | torch.Tensor, bias: torchtree.core.abstractparameter.AbstractParameter | torch.Tensor | None = None, cache_size=0)[source]#

Bases: torch.distributions.Transform

Transform via the mapping \(y = xA' + b\).

Example:

>>> x = torch.rand(3,2)
>>> weight = torch.rand(5, 2)
>>> bias = torch.rand(5)
>>> transform = LinearTransform(weight, bias)
>>> y = transform(x)
>>> torch.all(y == torch.nn.functional.linear(x, weight, bias))
