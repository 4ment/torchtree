ScaleMixtureNormal

Scale mixture of Normal distributions.
class torchtree.distributions.scale_mixture.ScaleMixtureNormal(id_: torchtree.typing.ID, x: torchtree.core.abstractparameter.AbstractParameter, loc: torchtree.core.abstractparameter.AbstractParameter | float, scale: torchtree.core.abstractparameter.AbstractParameter, gamma: torchtree.core.abstractparameter.AbstractParameter, slab: torchtree.core.abstractparameter.AbstractParameter | float = None)#

Scale mixture of Normal distributions.

Regularized when a slab width parameter or scalar is provided [piironen2017]

Parameters:

  • id – ID of object

  • loc – mean of the distribution

  • x – random variable

  • scale – global scale

  • gamma – local scale

  • slab – slab width

[piironen2017]

Piironen and Vehtari 2017. Sparsity information and regularization in the horseshoe and other shrinkage priors.

classmethod from_json(data: dict[str, Any], dic: dict[str, torchtree.core.identifiable.Identifiable]) → ScaleMixtureNormal#

Creates a ScaleMixtureNormal object from a dictionary.

Parameters:

  • data (dict[str, Any]) – dictionary representation of a ScaleMixtureNormal object.

  • dic (dict[str, Identifiable]) – dictionary containing torchtree objects keyed by their ID.

JSON attributes:

Mandatory:

  • id (str): unique string identifier.

  • x (dict or str): parameter.

  • loc (dict or str or float): location parameter.

  • scale (dict or str): global scale parameter.

  • local_scale (dict or str): local scale parameter.

Optional:

  • slab (dict or str or float): slab parameter

Example:

>>> x = {"id": "x", "type": "Parameter", "tensor": [1., 2.]}
>>> loc = {"id": "loc", "type": "Parameter", "tensor": [1.]}
>>> global_scale = {"id": "global", "type": "Parameter", "tensor": [1.]}
>>> local_scale = {"id": "local", "type": "Parameter", "tensor": [0.1, 0.2]}
>>> mixture_dic = {"id": "mixture", "x": x, "loc": loc,
...     "global_scale": global_scale, "local_scale": local_scale}
>>> mixture = ScaleMixtureNormal.from_json(mixture_dic, {})
>>> isinstance(mixture, ScaleMixtureNormal)
True
>>> isinstance(mixture(), Tensor)
True
