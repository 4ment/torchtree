torchtree.evolution.io.split_newick(newick: str) → list[str][source]#

Split tree in newick format around (),;

Example

>>> newick = '((a:1[&a={1,2}],b:1):1,c:1);'
>>> split_newick('((a:1,b:1):1,c:1);')
['(', '(', 'a:1', ',', 'b:1', ')', ':1', ',', 'c:1', ')', ';']
Parameters:

newick (str) – newick tree

Return List[str]:

list of strings
